# 🌿 Plant Disease Recognition using IPCV-Enhanced Deep Feature Fusion (MATLAB Project)

## 📌 Overview

This project is a **Plant Disease detection system** developed using MATLAB.
It helps in detecting plant diseases using image processing techniques and provides insights about plant health.

---

## 🚀 Features

* 🌱 Plant disease detection using images
* 🧠 MATLAB-based processing and analysis
* 📊 Pre-trained models using `.mat` files
* 🖥️ GUI interface using MATLAB (`.fig` + `.m`)

---

## ⚙️ How to Run

1. Download the dataset and `.mat` files from the link above
2. Place them in the project root folder (or as required)
3. Open MATLAB
4. Run trainCNN_Attention.m file in Code/
5. Then run finalPipelineCNN.m or PlantGUI.m

```
PlantGUI.m
```
